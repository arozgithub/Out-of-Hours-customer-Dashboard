# Project Summary
The Out-of-Hours Customer Service Portal for JobLogic is a specialized platform designed to enhance customer support during off-business hours. It features a comprehensive Master Dashboard for tracking all customer jobs and a Customer Dashboard for detailed job management, including real-time engineer statuses and customizable alerts. This portal streamlines job management, ensuring timely responses to customer inquiries, thereby improving customer satisfaction and operational efficiency.

# Project Module Description
- **Master Dashboard**: Central interface displaying all jobs across customers with search functionality, job statistics, and an end-of-shift report.
- **Customer Dashboard**: Detailed view for individual customers, showcasing job statuses, engineer information, and quick access to log new jobs.
- **Job Logging Wizard**: A step-by-step wizard for logging new jobs, with improved validation and dynamic SLA calculations.
- **Customer Alerts Portal**: Interface for managing alerts, allowing users to set notifications for job status updates.
- **Job Alerts Page**: Displays jobs logged against a specific customer with detailed tables of job statuses.

# File Description Inventory
- **src/pages/Index.tsx**: Main application routing and state management, handling customer data flow to the Job Log Wizard.
- **src/components/MasterDashboard.tsx**: Displays all jobs and statistics for the Master Dashboard.
- **src/components/CustomerDashboard.tsx**: Displays job details for a specific customer, enhanced with engineer availability and job status tracking.
- **src/components/JobLogWizard.tsx**: Step-by-step wizard for logging new jobs, recently fixed to resolve syntax errors and enhance functionality.
- **src/components/CustomerAlertsPortal.tsx**: Interface for managing customer job alerts.
- **src/components/JobAlertsPage.tsx**: Displays jobs for specific customers with detailed statuses.
- **src/components/JobCard.tsx**: Displays individual job details and manages job status updates.
- **src/components/StatusBadge.tsx**: Visual indicator for job status.
- **src/lib/jobUtils.ts**: Utility functions for job management, including status updates and mock data generation.
- **src/types/job.ts**: TypeScript interfaces for job and customer data.
- **src/components/JobEditModal.tsx**: Modal for editing job details including status, priority, contact info, and SLA times.
